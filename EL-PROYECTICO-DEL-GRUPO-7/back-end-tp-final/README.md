## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Instalación NestJS

```bash
$ npm i -g @nestjs/cli
$ nest new project-name
```

## ¿Cómo levantar el proyecto?

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Lo visualizamos en `http://localhost:3000/`
